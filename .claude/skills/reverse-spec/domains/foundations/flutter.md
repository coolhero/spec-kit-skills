# Flutter Foundation

## F0. Detection Signals

- `pubspec.yaml` with `flutter` SDK dependency
- `lib/main.dart` entry point
- `android/` and `ios/` directories
- `.dart` source files with Flutter widget imports
- `flutter` CLI commands in build scripts
